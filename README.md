# Farsi Quest

A mobile-friendly, static Farsi reading game for kids who already speak English
and Farsi and are learning to read Farsi letters and simple words.

The game is fully contained in `index.html` and is ready for GitHub Pages.

## GitHub Pages

This repo does not require Node.js or a build step to deploy.

To publish with GitHub Pages:

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to **Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Choose your branch and set the folder to `/ (root)`.
6. Save.

GitHub Pages will serve `index.html` directly.

## Local Preview

You can open `index.html` directly in a browser, or serve the folder locally:

```bash
python -m http.server 8080
```

Then open:

```text
http://localhost:8080/
```
